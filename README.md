# VerbalBridge - AI Speech Regeneration (Flutter Edition)

**Adaptive Clinical Suite for Aphasia Rehabilitation**

VerbalBridge is a clinical-grade Flutter application developed for the AIISH Hackathon. It bridges the communication gap for persons with Aphasia through evidence-based modules like WAB-R assessment, CILT, and Adaptive Cueing hierarchies.

## 🚀 Month 1 Core Features
- **Core Assessment Tool**: Real-time WAB-R scoring with a professional Clinical Slate UI and dynamic AQ calculation.
- **CILT Module**: Constraint-Induced Language Therapy with forced verbal cueing and speech recognition state management.
- **Aphasia Engine**: Robust static analysis engine for instant clinical classification (Broca, Wernicke, Global, etc.).
- **Adaptive Cueing**: Smart Semantic-Phonological hierarchy for progressive rehabilitation.

## 🛠 Tech Stack
- **Framework**: Flutter (Dart)
- **State Management**: Provider
- **UI Design**: Material 3 with "Clinical Slate" palette.
- **Backend**: Firebase (Auth, Firestore).

## 🏃 Getting Started
1. `npm install`
2. `npm run dev`
